# DESCRIPTION
CEFAT BUSINESS SCHOOL est une école de supérieure privée située.

# OBJECTIF
Créer un site web complet (Accueil, Formations, À propos, Contact, etc.) pour CEFAT BUSINESS SCHOOL. Les fonctionnalités seront les suivantes : pré-inscription, espace étudiants et intégration de moodle.

# STACK
- Next JS 16 avec App Router et TypeScript en mode strict
- Tailwind CSS
- Supabase pour la base de données et l'authentification et le stockage des fichiers (Nous utilisons la version locale)
- Vercel pour le déploiement
- Git pour le contrôle de version
- pnpm pour le gestionnaire de paquets

Les CLI de supabase et vercel sont installés globalement et seront utilisés pour les commandes principales.
